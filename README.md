Thaitones
=========

Thaitones lets you add tone diacritics to Thai transcription using the following upper-case letters:
M - Mid tone (no diacritic)
L - Low tone
F - Falling tone
H - High tone
R - Rising tone

The letter can be placed after the vowel or syllabel.

Examples
--------
- BpenM yaangL raiM baangF => Bpen yàang rai bâang
- BpeMn yaaLng raiM baaFng => Bpen yàang rai bâang
- maH-haaR witH-taH-yaaM-laiM => má-hǎa wít-tá-yaa-lai
- MaHhaaRwitHtaHyaaMlaiM => Máhǎawíttáyaalai

You can use directly at [thaitones.github.io](http://thaitones.github.io).
